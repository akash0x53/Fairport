00043 // Many of the low-level data structures in Fairport rely on being laid out
00044 // in the fashion preferred by Visual C++.  For example, these structures
00045 // need to align ulonglong to the nearest 8 bytes.  Under GCC, we can force
00046 // this layout behavior on Mac and Linux systems by appending an
00047 // appropriate attribute declaration to the structure.  We're also tried
00048 // '#pragma ms_struct on', but it fails on at least some Linux systems.
